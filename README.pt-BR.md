# Análise Inteligente de Logs para Detecção de Comportamentos Suspeitos

Este projeto propõe o desenvolvimento de um sistema inteligente para análise de logs de sistemas computacionais, com o objetivo de identificar comportamentos suspeitos utilizando técnicas de aprendizado de máquina.

## Motivação
Ambientes computacionais modernos geram grandes volumes de logs diariamente, tornando a análise manual inviável e suscetível a erros. Automatizar esse processo é essencial para apoiar a detecção de incidentes de segurança de forma mais eficiente.

## Objetivos

### Objetivo Geral
Desenvolver um sistema capaz de detectar comportamentos suspeitos em logs de sistemas utilizando técnicas de inteligência artificial.

### Objetivos Específicos
- Coletar e preparar logs de segurança  
- Implementar uma abordagem baseada em regras como baseline  
- Aplicar algoritmos de aprendizado de máquina para detecção de anomalias  
- Comparar os resultados entre métodos tradicionais e métodos baseados em IA  

## Tecnologias Utilizadas
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

## Estrutura do Projeto
- `data/`: dados brutos e processados  
- `src/`: código-fonte do sistema  
- `notebooks/`: análises exploratórias  
- `results/`: resultados e métricas  

## Metodologia
1. Revisão bibliográfica  
2. Pré-processamento dos logs  
3. Extração de características  
4. Implementação de regras heurísticas  
5. Aplicação de modelos de aprendizado de máquina  
6. Avaliação e comparação dos resultados  

## Trabalhos Futuros
- Análise em tempo real  
- Suporte a diferentes tipos de logs  
- Integração com ferramentas SIEM
